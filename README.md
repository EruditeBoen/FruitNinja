### FruitNinja with Handtracking
## Versioning

* python: 3.9.13 (>3.11 not supported)
* mediapipe: 0.10.21
* cvzone: LTS

## Virtual Environment (if needed)
```bash
python -m venv [name]
```
Replace [name] with the name of the environment

## Activiating venv
# Windows
```bash
[name]/Scripts/activate
```

# WSL (linux)
```bash
source [name]/bin/activate
```

## Libraries
```bash
pip install cvzone
pip install mediapipe==0.10.21
```

## Running

```bash
python main.py
```
